Guia Interativo de Flexbox

Explore as principais propriedades do Flexbox com exemplos visuais. Cada seção abaixo demonstra um conceito diferente, aplicado a um container cinza e seus itens azuis.

1. Direção dos Itens (flex-direction)

Define a direção em que os itens são organizados. O padrão é row (linha).

flex-direction: row; (Padrão)

4. Crescimento do Item (flex-grow)

Permite que um item "cresça" para ocupar o espaço livre no container. É uma propriedade do item, não do container.

Abaixo, o item 2 tem flex-grow: 2 e os outros têm flex-grow: 1. Ele ocupará o dobro do espaço livre que os outros.

Item 2 (grow: 2)
Item 3 (grow: 1)

5. Quebra de Linha (flex-wrap)

Por padrão, os itens tentam caber em uma única linha. Com flex-wrap: wrap;, eles quebram para a linha seguinte quando não há mais espaço.

6. Ordem dos Itens (order)

Muda a ordem visual de um item sem alterar o HTML. Por padrão, todos têm order: 0. Números menores vêm primeiro.

No HTML: Item A, Item B, Item C. Visualmente, a ordem é B, A, C.

Item A (order: 2)
Item B (order: 1)
Item C (order: 3)
